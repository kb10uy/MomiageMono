# Momiage Mono
Coding font for sidelock lovers!

## 概要
Momiage Mono は、以下のフォントからなるコーディング用の等幅フォントです。
* 欧文: [JetBrains Mono](https://www.jetbrains.com/ja-jp/lp/mono/)
* 日本語: [源暎モノゴ](https://okoneya.jp/font/genei-mono-go.html)
